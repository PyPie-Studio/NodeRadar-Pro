# Contributing to NodeRadar Pro

Thank you for your interest in **NodeRadar Pro**! We appreciate the enthusiasm and support from the network engineering community. 

Because NodeRadar Pro is a **closed-source freeware** project developed by **PyPie Studio**, the contribution process differs from traditional open-source repositories. While we do not accept direct code contributions or Pull Requests, your feedback and insights are vital to making this the best network reconnaissance tool on the market.

---

## 🤝 How You Can Contribute

Since the source code is not public, contributions focus on identifying issues, suggesting improvements, and helping other users.

### 1. Reporting Bugs
If you encounter a crash or a logical error, please open an **Issue** on this repository. To help us resolve it quickly, include:
- **Reproducibility:** Step-by-step instructions on how to trigger the bug.
- **Environment:** Your Operating System version (e.g., Windows 11 23H2) and network topology.
- **Logs:** Attach the contents of your `crash.log` located in `Documents\PyPie Studio\NodeRadar Pro\logs\`.
- **Integrity Hash:** Mention if the "Integrity Shield" in the System Logs shows any mismatch.

### 2. Feature Requests
We are always looking to expand NodeRadar Pro's capabilities. If you have an idea for a new probe, UI refinement, or integration:
- Open an **Issue** with the `enhancement` label.
- Explain the use case: How would this feature help a Senior IT Admin or Sysadmin?
- Provide a rough mockup or technical description if applicable.

### 3. Localization
While the current UI is English-only, we are considering adding support for other languages. If you are interested in translating the app's string resources, please reach out to us via email.

---

## 🏗️ Pull Request Policy

As this repository serves as a public landing page and issue tracker for compiled software:
- **Pull Requests containing code will be closed.**
- Pull Requests for **documentation** (improving `README.md`, fixing typos, adding FAQ items) are welcomed and encouraged.
